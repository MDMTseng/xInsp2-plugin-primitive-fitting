Polynomial (filter & primitive)

`poly_degree` is shared: as residual filter after Line/Circle/Ellipse fits, OR as the primitive degree when Fit Model is Polynomial. `poly_max_slope_deg` caps the curve's steepness (Polynomial fit only; 0 = no limit).

Learned Peak Filter (optional CNN)

When enabled, replaces the per-caliper gradient + NMS peak detector with a 1-D CNN forward pass over all calipers. Requires an ONNX file produced by lab/cnn (input [B,3,80] → output [B,80]). Reuses Min edge strength (0–255) as the probability threshold × 255. Falls back to classical extraction on load failure.
